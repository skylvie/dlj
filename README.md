# dlj
A CLI for downloading everything you can on [PKGj](https://github.com/blastrock/pkgj)

## Features
- Browser catalog & download packages for the following:
    - PS Vita Games
    - PS Vita DLCs
    - PS Vita Demos
    - PS Vita Themes
    - PS1 Games
    - PSP Games
    - PSP DLCs
    - PSM Games

## Installation
```sh
$ npm i dlj
```
Or if you want to run from source:
```sh
$ git clone https://github.com/skylvie/dlj
$ cd dlj
$ bun i
$ bun run src/cli.ts
```
